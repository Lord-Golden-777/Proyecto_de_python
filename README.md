Proyecto de Python

Proyecto final de mi curso de Python básico – Warders

Este es uno de mis primeros proyectos programando en Python.

¿De qué trata?

El juego trata de un caballero que debe atravesar 10 pisos de una torre llena de monstruos que intentarán detenerlo.

¿Qué aprendí?

Uso de variables

Bucles
Funciones
Pygame básico

Uso del teclado y el mouse
Lógica de programación
Conocimiento básico de Pygame

Mi proceso

El código lo hice parcialmente a mano y parcialmente con ayuda de IA, intentando aplicar todo el conocimiento que adquirí en el curso de Python básico. Tomé assets de Pinterest y los utilicé en mi juego.

Originalmente quería que la torre tuviera 100 pisos, pero debido a la gran cantidad de bugs, me vi obligado a usar IA para intentar solucionarlos. Aun así, no pude implementar todos los assets ni todas las ideas, principalmente por falta de tiempo y por los errores del juego.

A pesar de todo, gané el primer puesto en mi curso de Python básico entre todos mis compañeros.

¿Cómo se juega?
Usas las flechas del teclado para moverte.
Muévete directamente hacia los monstruos para atacar automáticamente; tanto tú como el enemigo recibirán daño.
Tienes una barra de vida que se agotará al atacar enemigos.
Los monstruos pueden soltar vida o espadas que puedes recoger para aumentar tu daño y vida.
Si recoges vida cuando ya tienes la barra al máximo, en su lugar obtendrás escudo.
En cada piso hay una probabilidad de que aparezca un cofre; haz clic para abrirlo. Te dará vida y escudo máximos.
Si recibes demasiado daño al atacar monstruos, morirás y el juego se detendrá.
